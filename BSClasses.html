Bootstrap Grid System

It helps to create different col sizes
Number of Cols can be vary with the different screen sizes
Bootstrap uses 12 cols grid system in a row.

Grid System Classes

Equal Col sizes using BS Break Points(Screen Sizes)

  • .col-xl
  • .col-lg
  • .col-md
  • .col-sm
Note: If we use md class, Number of n cols will be same from md to xl, But whenever screen size will be lower than medium number of n cols will be reduced automatically

Diff Col Sizes in all Screen sizes

  • .col-* (* = 1 to 12)
